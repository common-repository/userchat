UserChat - Чат через мессенджеры.

== Description ==

Плагин добавляет полученный код из сайта <a href="https://userchat.online" target="_blank" rel="nofollow">Userchat - Чат с клиентами через мессенджеры</a> на ваш сайт.

== Installation ==

1. Скопируйте папку плагина <strong>userchat</strong> в <strong>/wp-content/plugins/</strong>. 
2. Активируйте плагин через меню <strong>Плагины</strong>.
